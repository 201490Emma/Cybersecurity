Safe System

A safe system is very important, when you're using a computer or laptop. There are several and different ways to keep your system safe, such as:

  • always updating
  • always backing up
  • secure and lock the device
  • virus scanners

Why do you need to update?

Updating softare is not only suseful to improve the security. Moreover, the most recent software often works better with the new decives and technique. So use of all those modern features of the latest software release and download the most recent updates.

Why do you need to make a back- up?

People make mistakes more often than you think. Expect e- mails with viruses, it is also possible that important data is accidentally deleted. This can easily be solved by making a backup once in a while. You can easily restore a system with the push of a button!

Why do you need to secure and lock the device?

If the memory storage device or harddrive is stolen, it is possible that data loss can be prevented by encrypting the data on thedevice such as address book,authentication data and files. Once encryption is enabled, all the future data on the device will be encrypted safed. This prevents other people from easily enteringyour harddrive.
