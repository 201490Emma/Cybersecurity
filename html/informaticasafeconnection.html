Safe Connection

Internet Banking, sending a mail, using Facebook and so on are websites where you log in. You dont want your information to be intecepted. Make sure you're logging in to a website that has safe connection.

You can recognize a safe connection by a lock in the adress bar. You can get the lock in the adress bar from the web browser, if ou have a connection that is secures with a SSL certification. SSl stands for Secure Sockers Layer and it means that there is a secured layer that's placed between a server and a internet browser. It makes sure that the information is secured. It doesn't means that you can trust the website self.

Watch out on which WiFi your are on. Some connections can transfer viruses. WiFi is the commercial name for W- Lan ( Wireless Local Area Network ) The name WiFi, also written as Wi-Fi, is related to the well- known term Hi- Fi from the audio world. Hi- Fi stands for High Fiddelity. That's why the term Wireless Fidelity is also used. You also have hotspots, which you must secure properly, because you don't want random people using your hotspot. When you're using hotspot, know from who or what it is, because it can give you viruses.
